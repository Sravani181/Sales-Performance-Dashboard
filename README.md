# Sales Performance Dashboard

## Project Overview
This project analyzes sales data using Excel, MySQL, and Power BI.

## Tools Used
- Microsoft Excel
- MySQL
- Power BI

## Features
- Data Cleaning
- Pivot Tables
- SQL Analysis
- Interactive Power BI Dashboard
- Business Insights

## Business Insights
- West region generated the highest sales.
- Technology category achieved the highest sales and profit.
- Standard Class had the highest sales among shipping modes.
- Consumer segment purchased the highest quantity.
- New York City generated the highest sales.
